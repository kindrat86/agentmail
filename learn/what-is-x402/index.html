What is x402?

Also known as: HTTP 402 payments

x402 is a protocol pattern that uses the HTTP 402 'Payment Required' status to let clients — including autonomous AI agents — pay for API access or resources programmatically. Because x402 enables agents to move money without a human in the loop, sanctions screening in the payment path becomes essential.
