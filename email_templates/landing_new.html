FAQ

Does this work with any agent framework?

Yes. MCP tools for Claude Code, Cursor, and Windsurf. HTTP API for any language. CLI for your terminal.

What if my agent uses x402?

agentmail accepts x402 payments for compliance checks. Your agent pays $0.01 USDC on Base, gets the sanctions result, proceeds only if clean. See the x402 flow.

How up to date is the OFAC data?

Synced hourly from the US Treasury SDN list. If the Treasury adds a wallet at 2:47 PM, agentmail catches it at 3:00 PM.

Can I self-host?

Yes. MIT licensed. pip install sanctions-mcp and run your own instance. Full data, zero leakage.
