The flow — 4 steps your agent understands

Agent calls without payment

No API key. No auth. Just the endpoint.

2

Server returns 402: "Pay $0.05 USDC"

Payment requirements: network, asset, amount, pay_to wallet.

Agent pays via x402 facilitator

Agent FAQ

Does my agent need a wallet on Base?

For x402, yes. Any Base wallet with USDC balance works.

Can I call this from any agent framework?

Yes. Any agent that speaks HTTP can use x402. LangChain, CrewAI, Eliza, or raw Python.

What happens if the payment fails?

Server returns 402 again. No payment, no check. Retry with a new payment.

How current is the OFAC data?

Synced hourly from the US Treasury SDN list.

Can I self-host?

Yes. MIT licensed. pip install sanctions-mcp.
