What happens if an AI agent payment hits a sanctions match?

Answer: SanctionsAI returns a blocked transaction response with the specific sanctions list entry, match confidence score, and audit trail — the payment never executes, and the compliance team gets an instant alert with full context for regulatory reporting.

The blocked payment flow: match detected → transaction frozen (funds never leave the wallet) → alert generated (Slack/email/webhook with full match details) → compliance review dashboard (human reviews match, can override if false positive) → audit trail generated (OFAC-compliant record of the screening decision).
