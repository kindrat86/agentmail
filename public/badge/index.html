How it works

The badge checks the embedding page's domain against OFAC SDN, EU, UN, and UK sanctions lists. If the domain or associated entities appear on any list, the badge shows a warning. If clean, it shows "Verified Clean" with a timestamp.

No API key needed. No JavaScript. Pure server-side check via the iframe.
